# 🧠 CodeSync

CodeSync is a collaborative web-based code editor powered by the Monaco Editor. It allows users to write, edit, and share code with real-time sync capabilities, supporting multiple languages and themes. Ideal for developers, students, and teams.

---

## 🚀 Features

- ✅ Monaco Editor with theming
- ✅ Language switching
- ✅ Persistent `localStorage` saving
- ✅ Clerk authentication
- ✅ Convex backend integration
- 🔜 Real-time collaboration (WebSockets)
- 🔜 Code execution with output display

---

## 🛠️ Tech Stack

- **Frontend**: React + TailwindCSS + TypeScript
- **Editor**: Monaco Editor
- **Auth**: Clerk.dev
- **Backend**: Convex
- **Hosting**: Vercel
- **Others**: Lucide icons, Zustand for state management
